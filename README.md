# CityCare

Welcome to the CityCare repository! This project contains our ER diagrams, workflow flowcharts, user stories, and frontend web files. 

## 🚀 Getting Started for Teammates

To contribute to this project, you will need to clone this repository to your local machine.

### Prerequisites
Make sure you have [Git](https://git-scm.com/downloads) installed on your computer.

### 1. Clone the Repository
Open your terminal (or Git Bash) and run the following command to download the project files:

```bash
